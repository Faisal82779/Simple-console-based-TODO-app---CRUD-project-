# 📚 Todo Application — Console-Based Task Management System

## 🚀 Project Overview

This project features a comprehensive task management application built with a secure relational database architecture. It provides a complete Command Line Interface (CLI) workflow handling multi-user environments with robust **User Authentication (Registration & Login)** and full **Task CRUD operations (Create, Read, Update, Delete)**.

The system incorporates data validation layers preventing malformed data entry, dynamic relational mapping via user-to-task bindings using **Sequelize ORM**, and a structured search engine for high-efficiency keyword querying. Every core operation is backed by explicit business logic guardrails to ensure database integrity and seamless session state transitions.

Strict conditional logic and exception handling are embedded within every application workflow to guarantee application stability and faultless execution..

---

## 🛠️ Technology Stack

| Tool / Technology | Purpose |
|------|---------|
| **JavaScript (Node.js)** | Core programming language and runtime environment for application architecture, workflow logic, and CLI menus |
| **Sequelize ORM** | Object-Relational Mapper used for modern, secure, and structured data handling, model creation, and database relationships |
| **MySQL Database** | Persistent relational data storage for normalized user and task schemas |
| **DBeaver** | Database Administration Tool / Graphical user interface (GUI) for database design, query execution, and management |
| **mysql2 / Driver** | Core database connector enabling Sequelize to securely communicate with the MySQL database instance |
